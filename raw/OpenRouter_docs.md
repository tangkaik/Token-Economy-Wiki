# OpenRouter - Unified API for AI Models

> 来源：https://openrouter.ai/docs/quickstart
> 截图：已保存

---

## 核心定位

OpenRouter provides a unified API that gives you access to hundreds of AI models through a single endpoint, while automatically handling fallbacks and selecting the most cost-effective options.

---

## 三种接入方式

| 方式 | 适用场景 |
|------|----------|
| **API** | Full control, any language, no dependencies |
| **Client SDKs** | Type-safe model calls with minimal overhead |
| **Agent SDK** | Building agents with tool use, loops, and state |

---

## 主要功能

### Models & Routing
- **Model Fallbacks**：自动回退到其他模型
- **Provider Selection**：选择特定provider
- **Auto Exacto**：自动精确匹配
- **Model Variants**：模型变体支持
- **Routers**：路由器功能

### Features
- **Workspaces**：工作空间
- **Presets**：预设配置
- **Response Caching**：响应缓存
- **Tool Calling**：工具调用
- **Server Tools**：服务器工具
- **Plugins**：插件系统
- **Structured Outputs**：结构化输出
- **Message Transforms**：消息转换
- **Zero Completion Insurance**：零完成保险
- **ZDR**：Zero Data Retention
- **App Attribution**：应用归属
- **Guardrails**：护栏
- **Service Tiers**：服务层级
- **Input & Output Logging**：输入输出日志
- **Broadcast**：广播功能

### Privacy & Best Practices
完整的隐私政策和最佳实践指南

## 关键特性

1. **统一API**：一个endpoint接入数百模型
2. **自动路由**：自动选择最便宜的选项
3. **Model Fallbacks**：主模型不可用时自动回退
4. **Client SDKs**：支持Python、JavaScript等多种语言
5. **Agent SDK**：专为AI Agent设计，支持tool use、loops、state
