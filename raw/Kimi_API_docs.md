## 主要功能

### 模型能力
- **多模态理解**：支持文本、图片与视频输入
- **长上下文**：支持256K上下文长度
- **思考模型**：Kimi K2 thinking model
- **联网搜索**：官方联网搜索工具

### 接入方式
- **API接入**：通过API调用模型能力
- **开发工作台**：https://platform.kimi.com/playground
- **控制台**：https://platform.kimi.com/console
