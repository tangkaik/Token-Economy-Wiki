## 模型概览

Claude 3.7 Sonnet - 最新旗舰模型

## 主要功能

### 模型能力
- **扩展思考（Extended Thinking）** - 支持长思考链
- **自适应思考（Adaptive Thinking）** - 根据任务自适应
- **结构化输出** - JSON/结构化输出支持
- **多语言支持** - 全球语言支持
- **嵌入（Embeddings）** - 向量嵌入能力
- **PDF支持** - 文档处理
- **视觉理解** - 图像输入支持

### 工具（Tool Use）
- **网页搜索工具** - 实时联网搜索
- **网页抓取工具** - 抓取网页内容
- **代码执行工具** - 安全执行代码
- **计算机使用工具** - 自动化操作
- **Bash工具** - 命令行操作
- **文件编辑工具** - 文本编辑

### 上下文管理
- **上下文窗口** - 大上下文支持
- **提示缓存** - 降低重复调用成本
- **上下文编辑** - 编辑对话上下文
- **Token计数** - 精确计量

### 高级功能
- **Agent构建** - 使用Messages API和工具构建Agent
- **MCP支持** - Model Context Protocol
- **批量处理** - 批量API调用
- **搜索结果** - 搜索增强生成
